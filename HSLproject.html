HSL website usability assesment

User Experience | School Project

A solo school project on user experience and research methods.

My tasks included:
Identifying usability issues with the HSL website using hierarchical task analysis (HTA).
Conducting an experiment on a classmate about if their behaviour would differ from my assumptions on a HTA-model.
Analyzing experiment results using the ABCS framework (Ritter 2014) and Ceaparu's (2004) theory on user frustration.
Design my own prototype with Figma that resolves the most glaring issue, that I tought was accessability for non-finnish-speaking users. Also disabled people visiting Helsinki can have hard time in finding the information on which trams are accessible for example.
Then justifying the changes by leaning on the frameworks mentioned earlier.
